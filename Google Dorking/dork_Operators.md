| Operator      | Purpose                                      | Example                       |
| ------------- | -------------------------------------------- | ----------------------------- |
| `site:`       | Search only within a specific website/domain | `site:example.com login`      |
| `filetype:`   | Search specific file extensions              | `filetype:pdf cybersecurity`  |
| `ext:`        | Same as `filetype:`                          | `ext:xlsx salary`             |
| `intitle:`    | Word must appear in page title               | `intitle:"index of"`          |
| `allintitle:` | All words must appear in title               | `allintitle:admin login`      |
| `inurl:`      | Word must appear in URL                      | `inurl:admin`                 |
| `allinurl:`   | All words must appear in URL                 | `allinurl:admin login`        |
| `intext:`     | Word must appear in page content             | `intext:"password reset"`     |
| `allintext:`  | All words must appear in content             | `allintext:username password` |
| `cache:`      | View Google's cached version                 | `cache:example.com`           |
| `related:`    | Find similar websites                        | `related:github.com`          |
| `define:`     | Get definition of a word                     | `define:DNS`                  |
| `info:`       | Show information about a website             | `info:example.com`            |
| `before:`     | Results before a date                        | `AI before:2024-01-01`        |
| `after:`      | Results after a date                         | `AI after:2024-01-01`         |
| `AROUND(X)`   | Search terms within X words                  | `linux AROUND(5) security`    |
